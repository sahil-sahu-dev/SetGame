# SetGame
Implementation of the card game 'Set'. Assignment 3 of the course CS193p (Developing Applications for iOS using SwiftUI), Stanford University. 


## Skills used
* Swift, SwiftUI
* MVVM
* Core Animations Framework (implicit + explicit)
* Shape protocol
